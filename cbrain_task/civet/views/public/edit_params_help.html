Civet Options

Not all options listed are provided on CBRAIN
To request additional options, leave feedback.

Online documentation is available here


Pipeline options
  • Restart from this stage: Restart the pipeline from the specified stage. This option appears only when restarting a task.
Volume options
  • Model: Define the model for stereotaxic registration [default: "icbm152nl" (MNI ICBM152 non-linear 6th generation) for version lower than 2.0.0 and icbm152nl_09s" (MNI ICBM152 non-linear symmetric (2009)) for version 2.0.0 and later, "icbm152nl_09a" (MNI ICBM152 non-linear asymmetric (2009)), "icbm152lin" (MNI ICBM152 linear), "ADNInl" (MNI ADNI non-linear, only for version 1.1.12), "ADNIhires" (MNI ADNI non-linear high resolution, only for version 2.0.0 and later)].
  • Model for surface registration: Define the model for surface registration [default: "icbm152MCsym" (ICBM152, marching-cubes, symmetric (2014)), "colinMCasym" (Colin, marching-cubes, asymmetric (2014)), "samirMCasym" (IBIS Phantom (/Samir), marching-cubes, asymmetric (2014))]. Only available for version 2.0.0 and later.
  • Template: Define the voxel size (in mm) for stereotaxic registration [default: 1.00; also available only for version 2.0.0 and later: 0.50, 0.75].
  • Degrees of freedom for linear registration: Number of parameters / degrees of freedom to be used in transformation for linear registration [default: 9; also available: 6, 12].
  • Interpolation: Interpolation method from native to stereotaxic space [default: trilinear; also available: tricubic, sinc].
  • Head height: Head height in mm for neck cropping. Suggested value: 170. Only available for version 1.1.12 and later.
  • Mask blood vessels: Mask blood vessels prior to white surface extraction. Only available for version 1.1.12 and later.
  • N3 distance: N3 spline distance in mm (suggested values: 200 for 1.5T scan; 50-125 for 3T scan. For older 3T scans, low values nearer 50 may work best; for newer 3T scans, high values nearer 125 may work best).
  • PVE: advanced = -correct-pve -subcortical -mask-cerebellum or classic = -no-correct-pve -no-subcortical -no-mask-cerebellum (default for version lower than 2.1.0).
Surface options
  • Do NOT build surfaces: Don't build surfaces, if checked.
  • High resolution surface: Build high resolution surfaces. Only available for version 2.0.0 and later.
  • Combine left/right surfaces: Combine left and right cortical surfaces, if checked.
  • Thickness method: Choose cortical thickness metric [default: tlink; also available: tlaplace, tfs (Freesurfer's metric, only for version 2.0.0 and later)], starting to version 2.1.0 you can select more than one method.
  • Thickness method for QC: Choose the cortical thickness that will be used for the QC, is accepted for version 2.1.0 and later.
  • Kernel size: Default: 20mm for version 1.1.11 and lower and 30mm for version 1.1.12 and later. List of integers separated by a ':' is accepted for version 2.0.0 and later, e.g. 10:20:30.
  • Resample surfaces: Resample cortical surfaces, if checked.
  • Surface parcellation: Use atlas for parcellation [default: lobar; also available: AAL, DKT-40 (only for version 2.0.0 and later)].
  • Kernel size areas: Blurring kernel size in mm for resampled surface areas. Default: 40mm. Only available for version 1.1.11 and later. List of integers separated by a ':' is accepted for version 2.0.0 and later, e.g. 10:20:30.
  • Kernel size volumes: Blurring kernel size in mm for resampled surface volumes. Default: 40mm. Only available for version 1.1.11 and later. List of integers separated by a ':' is accepted for version 2.0.0 and later, e.g. 10:20:30.
VBM options
  • Process VBM files: Process VBM files for analysis, if checked.
  • Blurring kernel size in mm for volume: Blurring kernel size in mm for volume.
  • Run symmetry tool: Run symmetry tools, if checked.
  • Mask cerebellum in VBM maps: Mask cerebellum in VBM maps, if checked.
ANIMAL options, only for version 2.0.0 and later
  • ANIMAL segmentation (non-linear lobe): Run volumetric ANIMAL segmentation [default -no-animal].
  • Lobe atlas for ANIMAL segmentation: Use lobe atlas for ANIMAL segmentation [default: icbm152nl-VI (ICBM152 generation VI symmetric model, use with "icbm152lin" and "icbm152nl" models, "icbm152nl-2009a" (ICBM152 2009a symmetric model, use with other model)" ].
MINC file input list options
  • Launch CIVET: This checkbox allows you to selectively enable or disable the processing of some files; useful when re-running partial sets of files.
  • Prefix: A simple name or identifier for a set of CIVET outputs. The same name must be used in order to combine several CIVET outputs into a CIVET study.
  • Subject ID: The ID of the subject being processed. These must all be distinct within each prefix.
